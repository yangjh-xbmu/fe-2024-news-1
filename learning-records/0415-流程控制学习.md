# JavaScript 流程控制学习记录

## 学习背景

- 零基础，做网页互动

## 一句话定义

流程控制决定代码「先走哪条路、后走哪条路」，包括顺序执行、分支、循环三种。

## 核心内容

### 分支控制 if / else

```js
let score = 75;
if (score >= 60) {
  console.log('及格');
} else {
  console.log('不及格');
}
```

### 三元运算符

```js
let score = 75;
score >= 60 ? console.log('及格') : console.log('不及格');
```

## 常见错误

| 错误 | 正确 |
|------|------|
| `if (age = 18)` 写成了单个 `=`（赋值） | `if (age === 18)` 用 `===` 比较 |
| `if` 后面不写 `{}` | 始终写 `{}`，养成习惯 |

## 练习答案

1. `75 >= 60` → 输出「及格」
2. `if (n === 0) { ... } else { ... }`
3. `score >= 60 ? console.log('及格') : console.log('不及格')`

## 陷阱题

```js
let x = 5;
if (x = 10) {  // 误写为赋值，x被赋值为10（truthy），永远走进上面的分支
  console.log('条件成立');
}
```
