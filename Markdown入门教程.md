# Markdown 10分钟入门指南

> 零基础快速上手，10分钟掌握核心语法

---

## 一、标题（2分钟）

```markdown
# 一级标题
## 二级标题
### 三级标题
#### 四级标题
##### 五级标题
###### 六级标题
```

> 规则：`#` 数量 = 标题级别，共支持6级

**效果预览：**

# 一级标题
## 二级标题
### 三级标题

---

## 二、文字格式（2分钟）

| 效果 | 语法 | 示例 |
|------|------|------|
| **粗体** | `**文字**` 或 `__文字__` | **粗体** |
| *斜体* | `*文字*` 或 `_文字_` | *斜体* |
| ***粗斜体*** | `***文字***` | ***粗斜体*** |
| ~~删除线~~ | `~~文字~~` | ~~删除线~~ |

## 三、列表（2分钟）

### 无序列表
使用 `-`、`*` 或 `+`：

```markdown
- 项目一
- 项目二
  - 子项目（前面加两个空格）
  - 子项目
- 项目三
```

**效果：**
- 项目一
- 项目二
  - 子项目
  - 子项目
- 项目三

### 有序列表
使用 `数字.`：

```markdown
1. 第一项
2. 第二项
3. 第三项
```

**效果：**
1. 第一项
2. 第二项
3. 第三项

---

## 四、链接与图片（2分钟）

### 链接
```markdown
[链接文字](https://example.com)
[链接文字](https://example.com "鼠标悬停显示的标题")
```

### 图片
```markdown
![图片描述](图片地址.jpg)
![图片描述](图片地址.png "图片标题")
```

### 自动链接
```markdown
<https://example.com>
<email@example.com>
```

---

## 五、代码（2分钟）

### 行内代码
使用单个反引号：
```markdown
这是 `行内代码` 的示例
```

### 代码块
使用三个反引号，可指定语言：

````markdown
```python
def hello():
    print("Hello, Markdown!")
```
````

```javascript
function hello() {
    console.log("Hello, Markdown!");
}
```

---

## 六、其他常用语法

### 引用块
```markdown
> 这是一段引用内容
> 多行引用
> > 嵌套引用
```

**效果：**
> 这是一段引用内容
> 多行引用
> > 嵌套引用

### 分割线
使用三个或以上的 `-`、`*` 或 `_`：

```markdown
---
***
___
```

### 表格
```markdown
| 表头1 | 表头2 | 表头3 |
|-------|-------|-------|
| 内容1 | 内容2 | 内容3 |
| 内容4 | 内容5 | 内容6 |
```

**效果：**

| 表头1 | 表头2 | 表头3 |
|-------|-------|-------|
| 内容1 | 内容2 | 内容3 |
| 内容4 | 内容5 | 内容6 |

## 单选题

### 第1题
**如何写出一个二级标题？**

A. `一级标题`

B. `## 二级标题`

C. `**二级标题**`

D. `-- 二级标题 --`

<details>
<summary>点击查看答案</summary>

**答案：B**

解析：`#` 表示一级标题，`##` 表示二级标题，以此类推，最多支持6级标题。

</details>

---

### 第2题
**如何让文字显示为粗体？**

A. `*文字*`

B. `~~文字~~`

C. `**文字**`

D. `` `文字` ``

<details>
<summary>点击查看答案</summary>

**答案：C**

解析：`*文字*` 是斜体，`~~文字~~` 是删除线，`` `文字` `` 是行内代码，`**文字**` 才是粗体。

</details>

---

### 第3题
**插入一个链接的正确语法是？**

A. `(链接文字)[https://example.com]`

B. `[链接文字](https://example.com)`

C. `![链接文字](https://example.com)`

D. `<链接文字|https://example.com>`

<details>
<summary>点击查看答案</summary>

**答案：B**

解析：`[文字](URL)` 是链接语法，`![文字](URL)` 是图片语法。

</details>

---

### 第4题
**创建有序列表应该使用？**

A. `- 项目`

B. `* 项目`

C. `+ 项目`

D. `1. 项目`

<details>
<summary>点击查看答案</summary>

**答案：D**

解析：`-`、`*`、`+` 都用于创建无序列表，数字加句点 `1.` 用于创建有序列表。

</details>

---

### 第5题
**以下哪种是代码块的正确写法？**

A. `` `代码` ``（单个反引号）

B. ```` ```代码``` ````（三个反引号）

C. `'代码'`（单引号）

D. `"代码"`（双引号）

<details>
<summary>点击查看答案</summary>

**答案：B**

解析：单个反引号是行内代码，三个反引号是代码块。单引号和双引号在Markdown中没有特殊含义。

</details>

<details>

斜体文字的写法是 `**文字**`。

<details>
<summary>点击查看答案</summary>

</details>

### 第8题
插入图片的语法是 `![图片描述](图片地址)`。

<details>
<summary>点击查看答案</summary>

</details>
